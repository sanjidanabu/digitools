# 🚀 DigiTools - Premium Digital Asset Marketplace

**DigiTools** is a modern, full-stack digital products marketplace and management system. Built with **Next.js**, **TypeScript**, and **Tailwind CSS**, it features a mobile-first responsive layout, smooth user interactions powered by **Framer Motion**, and a robust backend integrated with **MongoDB**.

---

## ✨ Features

*   **Live Product Grid & Search:** Dynamically fetches premium assets from MongoDB with real-time debounced search functionality and server-side pagination.
*   **Dynamic Asset Showcase:** Individual, SEO-friendly product details view for every digital template (`/products/[id]`).
*   **Item Management Panel (CRUD):** A secure dashboard for admins to list, view, and instantly delete digital products from the database.
*   **Adaptive & Responsive Layout:** Crafted with a mobile-first approach—displays beautifully as clean item cards on mobile screens and structural tables on desktops.
*   **Fluid Animations:** Seamless UI transitions and interactive hover elements using Framer Motion.
*   **Type-Safe Development:** Scalable codebase built entirely with strict TypeScript validation.
